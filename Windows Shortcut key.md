
## 🪟 Window & App Management

- **Alt + Tab** → Switch between open apps  
- **Alt + F4** → Close current app
- **Win + D** → Show / hide desktop
- **Win + L** → Lock your PC
- **Win + E** → Open File Explorer
- **Win + R** → Open Run dialog   
---
## 📂 File Explorer Power Moves

- **Ctrl + Shift + N** → New folder    
- **Alt + Enter** → File/Folder properties
- F2 → Rename selected file
- **Delete** → Delete to Recycle Bin
- **Shift + Delete** → Delete permanently

---
## 🔍 Search & System

- **Win + S** → Search (apps, files, settings)
- **Win + I** → Open Settings
- **Ctrl + Shift + Esc** → Task Manager
- **Win + Pause/Break** → System info
    

---

## 🧠 Productivity / Multitasking (Underrated but 🔥)

- **Win + Tab** → Task View (virtual desktops)  
- **Win + Ctrl + D** → New virtual desktop
- **Win + Ctrl + ← / →** → Switch desktops
- **Win + ↑ / ↓** → Maximize / Minimize window
- **Win + ← / →** → Snap window left/right

---

## 📸 Screenshots & Screen Tools

- **PrtScn** → Copy full screen
- **Alt + PrtScn** → Copy active window
- **Win + PrtScn** → Save screenshot automatically
- **Win + Shift + S** → Snipping Tool (best one!)

---
- Win + ; → Emoji
- Win + v → clipboard
